# Contributing

Mnemosyne Engine accepts contributions across code, graph data, policies, rubrics, and learning-object review.

## Standards

- Use shared schemas before adding new object shapes.
- Add every meaningful graph or governance change to an audit trail.
- Include sources for factual claims, or mark the claim as unsourced/personal.
- Do not merge high-stakes canonical content without the required review path.
- Prefer tests that measure behavior instead of snapshotting implementation details.

## Content Governance

Votes affect ranking, review priority, and escalation. Votes do not directly decide truth. Evidence, source quality, learning outcomes, AI audit, and human moderation all remain part of the decision record.
